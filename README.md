# Página Web para Mariachi Noche Mexicana

Este repositorio contiene el código fuente para la página web del Mariachi Noche Mexicana, una aplicación React moderna desarrollada con Vite y estilizada con Tailwind CSS. La página está diseñada para proporcionar información sobre el grupo, sus servicios, galería y facilitar el contacto con clientes potenciales.

## Arquitectura y Tecnologías

El proyecto utiliza React 19 con React Router para la navegación entre páginas, implementando una arquitectura de componentes modulares y reutilizables. Los componentes principales incluyen Header con navegación responsive, Hero section con gradientes animados, Footer informativo, y páginas dedicadas para Home, About, Services, Gallery y Contact. Todo el styling está implementado con Tailwind CSS v4, proporcionando un diseño consistente con colores temáticos mexicanos (tonos ámbar y dorado) y animaciones suaves.

La estructura de carpetas sigue las mejores prácticas de React, con componentes en `/src/components/`, páginas en `/src/pages/`, y configuración optimizada de Vite con el plugin oficial de Tailwind. El diseño es completamente responsive y incluye elementos de accesibilidad como aria-labels y indicadores de focus, manteniendo un rendimiento óptimo y una experiencia de usuario moderna.
